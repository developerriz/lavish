# LAVISH FURNITURE — Agent Rules

## Font
- Use **Poppins** as the only font. No other fonts allowed.

## Colors
- Black (#0a0a0a) for headings and primary elements.
- Teal (#00BFA5) as the accent color. Use sparingly.
- White (#ffffff) for all backgrounds.
- Neutral grays for body text and borders.

## Style Rules
- No rounded corners. All elements are square/sharp.
- No box shadows. Use borders instead.
- No gradients. Flat colors only.
- No dark mode (`dark:` prefix) classes.
- Bold, uppercase headings with tight tracking.
- Editorial, magazine-style layouts.
- Sections divided by thin top borders.
